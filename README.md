# Analyze-Earthquake-Logs

## Dependencies
* chromium-browser
* Python Library ```folium```

  ```bash
  pip3 install folium
  ```
## Details
First I Extracted Earthquakes Data, then Marking Coordinates on Map via Python Library ```folium``` and Show It in Web Page via ```chromium-browser```
